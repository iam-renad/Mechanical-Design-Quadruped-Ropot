# Mechanical-Design-Quadruped-Robot

Mechanical design and technical analysis task for an 8-DOF quadruped robotic prototype.

## Project Overview
This project focuses on the structural design, kinematics, and stability analysis of an 8-DOF quadruped robot. 
The design was developed using Tinkercad to ensure functional articulation and movement efficiency.
